# **H-News**

### **Proje Hakkında**

![App Screenshot](https://github.com/havaacigse/h-news/raw/main/news.jpg)

#### Proje Tanımı:

H-News, kullanıcıların güncel ve önemli haberlere kolayca erişebilmesini sağlayan modern bir mobil haber sitesidir. Kullanıcılar, uygulama üzerinden haberleri farklı kategorilere göre filtreleyebilir, favori haberlerini kaydedebilir ve internet bağlantısı olmadığında offline olarak okuyabilirler.
